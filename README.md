# Bihar Police SI Exam CBT

A Computer-Based Test (CBT) platform for Bihar Police Sub-Inspector (SI) exam preparation.

## Features

- 12 practice sets with 50 questions each
- Bilingual questions (English & Hindi)
- Real-time timer and question palette
- Instant result calculation with detailed review
- Test history tracking with view functionality
- Language toggle (English, Hindi, Both)
- Responsive design

## Getting Started

1. Open `si_login.html` in a web browser
2. Enter your roll number to start
3. Select a practice set from the dashboard
4. Complete the test within 60 minutes
5. View results and review answers
6. Check test history anytime

## Files

- `si_login.html` - Login page
- `si_paper_selection.html` - Practice set selection
- `si_exam.html` - Exam engine
- `si_result.html` - Result display
- `si_history.html` - Test history
- `siset1.js` to `siset12.js` - Question sets

## Scoring

- Correct answer: +2 marks
- Wrong answer: -0.2 marks
- Unanswered: 0 marks
- Total: 100 marks
